<task type="auto">
  <name>Task 2: Create SQLModel models with constraints and model tests</name>
  <files>
    app/models.py
    tests/__init__.py
    tests/conftest.py
    tests/test_models.py
  </files>
  <action>
    Create app/models.py:

    1. FIRST set SQLModel.metadata.naming_convention at module top (BEFORE any class):
       - ix: "ix_%(column_0_label)s"
       - uq: "uq_%(table_name)s_%(column_0_name)s"
       - ck: "ck_%(table_name)s_%(constraint_name)s"
       - fk: "fk_%(table_name)s_%(column_0_name)s_%(referred_table_name)s"
       - pk: "pk_%(table_name)s"

    2. Define BOARD_SIZE = 50

    3. Game model (table=True, __tablename__="game"):
       - id: int | None = Field(default=None, primary_key=True)
       - name: str
       - status: str = Field(default="setup")
       - created_at: datetime = Field(default_factory=datetime.utcnow)
       - CheckConstraint("status IN ('setup', 'playing', 'scoring', 'finished')", name="game_status_check")

    4. Player model (table=True, __tablename__="player"):
       - id: int | None = Field(default=None, primary_key=True)
       - game_id: int = Field(foreign_key="game.id")
       - name: str
       - color: str
       - score_total: int = Field(default=0)
       - turn_order: int
       - created_at: datetime = Field(default_factory=datetime.utcnow)
       - CheckConstraint("turn_order BETWEEN 1 AND 6", name="player_turn_order_range")
       - UniqueConstraint("game_id", "color", name="uq_player_game_color")
       - UniqueConstraint("game_id", "turn_order", name="uq_player_game_turn_order")
       - UniqueConstraint("game_id", "name", name="uq_player_game_name")
       - @property current_cell -> score_total % BOARD_SIZE
       - @property lap -> score_total // BOARD_SIZE

    5. ScoreAction model (table=True, __tablename__="score_action"):
       - id: int | None = Field(default=None, primary_key=True)
       - game_id: int = Field(foreign_key="game.id")
       - event_type: str
       - description: str | None = None
       - is_undone: bool = Field(default=False)
       - created_at: datetime = Field(default_factory=datetime.utcnow)
       - CheckConstraint with all 8 event types (ROAD_COMPLETED, CITY_COMPLETED, MONASTERY_COMPLETED, ROAD_FINAL, CITY_FINAL, MONASTERY_FINAL, FARM_FINAL, MANUAL), name="score_action_event_type_check"

    6. ScoreEntry model (table=True, __tablename__="score_entry"):
       - id: int | None = Field(default=None, primary_key=True)
       - action_id: int = Field(foreign_key="score_action.id")
       - player_id: int = Field(foreign_key="player.id")
       - points: int
       - score_before: int
       - score_after: int
       - UniqueConstraint("action_id", "player_id", name="uq_score_entry_action_player")

    Create tests/__init__.py (empty).

    Create tests/conftest.py:
    - Import app.db (to register the pragma event listener globally)
    - engine fixture: create_engine("sqlite://", connect_args={"check_same_thread": False}, poolclass=StaticPool), then SQLModel.metadata.create_all(engine), return engine
    - session fixture: Session(engine) yielded via context manager
    - game_with_players fixture: creates a Game(name="Test Game", status="playing"), flushes, creates Player Alice (blue, turn_order=1) and Player Bob (red, turn_order=2), commits, refreshes all, returns (game, [p1, p2])

    Create tests/test_models.py:
    - test_player_cell_zero: score_total=0 -> current_cell=0, lap=0
    - test_player_cell_mid: score_total=27 -> current_cell=27, lap=0
    - test_player_cell_one_lap: score_total=53 -> current_cell=3, lap=1
    - test_player_cell_three_laps: score_total=152 -> current_cell=2, lap=3
    - test_player_cell_exact_lap: score_total=50 -> current_cell=0, lap=1
    - test_game_status_constraint: insert Game with invalid status, assert IntegrityError
    - test_player_turn_order_range: insert Player with turn_order=7, assert IntegrityError
    - test_player_unique_color_per_game: insert two players with same color in same game, assert IntegrityError
    - test_player_unique_name_per_game: insert two players with same name in same game, assert IntegrityError
    - test_foreign_key_enforcement: insert Player with nonexistent game_id, assert IntegrityError (proves PRAGMA foreign_keys=ON)
    - test_score_entry_unique_action_player: insert two entries for same action_id+player_id, assert IntegrityError
  </action>
