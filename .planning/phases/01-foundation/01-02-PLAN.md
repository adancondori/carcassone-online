  <action>
    Implement all 4 service functions in app/services.py following the patterns from RESEARCH.md:

    **recalculate_score(session: Session, player_id: int) -> int:**
    - Use session.exec(select(func.coalesce(func.sum(ScoreEntry.points), 0))) with join on ScoreAction, filtered by player_id and ScoreAction.is_undone == False
    - Get player, set player.score_total = total, return total

    **add_score(session: Session, game_id: int, player_points: list[tuple[int, int]], event_type: str, description: str | None = None) -> ScoreAction:**
    - Create ScoreAction, add to session, flush to get id
    - For each (player_id, points) in player_points:
      - Get player via session.get(Player, player_id), raise ValueError if None
      - Record score_before = player.score_total
      - player.score_total += points
      - Create ScoreEntry with action_id, player_id, points, score_before, score_after=player.score_total
    - session.commit()
    - Return action

    **undo_last(session: Session, game_id: int) -> ScoreAction | None:**
    - Select last active ScoreAction (is_undone=False) for game_id, ordered by id DESC, limit 1
    - If None, return None
    - Mark is_undone = True
    - Get all ScoreEntry for this action, collect affected player_ids
    - For each affected player_id, call recalculate_score(session, pid)
    - session.commit()
    - Return the undone action

    **rollback_to(session: Session, game_id: int, action_id: int) -> int:**
    - Select all active ScoreActions for game_id where id > action_id
    - If empty, return 0
    - For each action: mark is_undone = True, collect affected player_ids from entries
    - For each affected player_id, call recalculate_score(session, pid)
    - session.commit()
    - Return count of undone actions

    Important: Use `session.exec(select(...))` (SQLModel style), NOT `session.query(...)` (deprecated).
    Use `func.coalesce` and `func.sum` from sqlalchemy for the aggregation query.
