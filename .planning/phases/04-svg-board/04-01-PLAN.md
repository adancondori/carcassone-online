       (528, 361),  # 0

   ]

2. Add a helper function `build_board_context(players)` in `app/web/dependencies.py` that:
   - Takes the list of player objects (already have score_total, color, name)
   - Groups players by cell (score_total % 50)
   - Computes stacking offsets using the same getStackOffset algorithm from the prototype:
     ```python
     import math

     def _stack_offset(index: int, total: int) -> tuple[float, float]:
         if total == 1:
             return (0.0, 0.0)
         angle = (2 * math.pi * index) / total - math.pi / 2
         radius = 13.0 if total <= 3 else 15.0
         return (math.cos(angle) * radius, math.sin(angle) * radius)
     ```
   - Returns a dict of `{cell_num: [{"cx": float, "cy": float, "hex": str, "color": str, "initial": str, "lap": int}, ...]}` for all occupied cells
   - Uses `PLAYER_COLORS[player.color]["hex"]` for the hex color
   - Uses `player.name[0]` for the initial letter

3. Update `_render_dashboard_fragments()` in `app/web/routes.py`:
   - Import `build_board_context` and `BOARD_CELLS` from dependencies
   - Add `board_cells = build_board_context(game_state.players)` 
   - Add `"board_cells": board_cells` to `base_context`
   - Add the board OOB fragment after action_bar_html:
     ```python
     board_html = templates.TemplateResponse(
         "dashboard.html",
         {**base_context, "oob": True},
         block_name="board",

     ```
   - Append `board_html` to the response content: `score_html + controls_html + action_bar_html + board_html`
